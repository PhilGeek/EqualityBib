EqualityBib by PhilGeek

Bibliography of philosophical papers on equality

A bibliography of relatively recent philosophical papers on equality compiled principally by Véronique Munoz-Dardé. The bibliography will be maintained separately from Philosophy.bib, though it will be merged into Philosophy.bib.

Dependencies

BibTeX

Most reference managers can import BibTeX files. JabRef is an open source cross-platform BibTeX front end. BibDesk is an open source BibTeX front end for OS X.
